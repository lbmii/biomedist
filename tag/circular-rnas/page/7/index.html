Circular RNAs

Web resources for functional studies of circular RNAs
With circBase, you can explore public circRNA datasets and download the custom python scripts needed to discover circRNAs in your own (ribominus) RNA-seq data. circBank is a comprehensive database for human circRNAs, where a novel naming system of circRNAs based on the host genes of circRNAs was implemented.
