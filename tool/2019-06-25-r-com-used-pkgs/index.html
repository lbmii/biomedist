R project and commonly used packages

Tuesday, 25 June 2019

R is a free software environment for statistical computing and graphics. It is the most widely used language for data science, together with python. The R Project for Statistical Computing provide multiple CRAN Mirrors to Download R. In China mainland, at least 10 mirror sites are available, such as those of USTC, LZU, SUSTech. Pick whatever one seems closest to you and start your work.

Rtools : Toolchains for building R and R package from source on Windows.

RStudio is also hepful and friendly supplement for R and can be downloaded by click the names.

RStudio Connect is the best way to host and share data products made with R and Python.

ggplot2 is a R package dedicated to create Elegant Graphics for Data Analysis. It can greatly improve the quality and aesthetics of your graphics, and will make you much more efficient in creating them. ggplot2 allows to build almost any type of chart ans is avaliable on GitHub as well.

Complex heatmaps are efficient to visualize associations between different sources of data sets and reveal potential patterns. Here the ComplexHeatmap package provides a highly flexible way to arrange multiple heatmaps and supports self-defined annotation graphics.

Bioconductor uses the R statistical programming language, and is open source and open development. It has two releases each year, and an active user community. Bioconductor is also available as Docker images. The mission of the Bioconductor project is to develop, support, and disseminate free open source software that facilitates rigorous and reproducible analysis of data from current and emerging biological assays. We are dedicated to building a diverse, collaborative, and welcoming community of developers and data scientists.

Biomedical Knowledge Mining using GOSemSim and clusterProfiler is meant as a guide for mining biological knowledge to elucidate or interpret molecular mechanisms using a suite of R package, including ChIPseeker, clusterProfiler, DOSE, enrichplot, GOSemSim, meshes and ReactomePA.

The WGCNA R software package is a comprehensive collection of R functions for performing various aspects of weighted correlation network analysis. The package includes functions for network construction, module detection, gene selection, calculations of topological properties, data simulation, visualization, and interfacing with external software.
