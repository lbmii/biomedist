Gene probes re-annotation with SeqMap and R (GPL10558)

Probes from ArrayChips released in GEO

Monday, 07 November 2022

Preparation Work:

Before starting for re-annotation, preparatory work includes:

  • Download probe annotation table from GEO (See how in Get GEO platform annotation file and import into R)
  • Download genome annotation file from GENCODE (See how in Download genome annotation files from GENCODE)
  • Download and unzip SeqMap Software for re-annotation
  • Install R
  • Install Git for Windows

Follow the preparation, create a new folder and name it with an English name without space is preferred. Then move probe annotation table, genome annotation file and SeqMap to the new-created folder.

Make sure that probe annotation table, genome annotation file and SeqMap are placed in the same folder.

Make sure that GPL10558_probe_seq.fasta is saved in the same folder with probe annotation table, genome annotation file and SeqMap.
  • Re-annotate the probes with SeqMap

Open Git Bash and change directory to the folder depositing GPL10558_probe_seq.fasta, probe annotation table, genome annotation file and SeqMap with command line.

cd E:
cd DataMining/RemapSeqMap/

Input the following command in Git Bash and press Enter.

./seqmap-1.0.12-windows.exe 0 ./GPL10558_probe_seq.fasta ./gencode.v42.transcripts.fa seqmap_results.txt //output_all_matches

Once finished, the re-annotation results will be save as seqmap_results.txt.

  • All names in the command must be the same with files in the folder!
  • Turn to SeqMap official site for more details.

# extract gene SYMBOL by self-defined function
def.extract <- function(x, sep = "//", element.order = 1){
    library(stringr)
    object_split <- str_split(x, sep)
    geneID <- sapply(object_split, "[", element.order)
    return(geneID)
  }

geneNames <- def.extract(seqmap.res$trans_id, "\\|", 6)
