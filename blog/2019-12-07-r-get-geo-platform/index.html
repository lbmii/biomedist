Get GEO platform annotation file and import into R

Saturday, 07 December 2019

Here is a demo with GEO Series of GSE50772.

# if not installed, install first
BiocManager::install("GEOquery")

# load bioconductor packages
library(GEOquery)
gseID <- "GSE50772"
platformID <- "GPL570"
  • Method 1: Download with GEOquery

Usually, we can get the annotation file automaticlly bu running code below:

gpl <- getGEO(platformID, destdir = ".")
It will download a file named GPL570.soft end with a suffix of soft.

You can find the file in the current working directory.

If failed, don’t be frustrated! We can download platform files manually and import it into R later.

Try as follows:

Search for GSE50772 in GEO and enter the series page. In the result page, places labelled with Platforms (1) (Arrowed 1) and Download family (Arrowed 2) can be found （Figure 1).

  • Method 2: Download annotation file manually of txt format
  1. Click the platform name GPL570 in Platforms (1) (Arrowed 1) in Figure 1 and scroll down to the bottom, as shown in Figure 2.
  1. Click Download full table... above Annotation SOFT table… (Arrowed in Figure 2) to start download.
  2. Import txt file into R with fread{data.table}
# install.packages("data.table")
library(data.table)
gpl <- fread("GPL570_55999.txt")
dim(gpl) # head(gpl), View(gpl), nrow(gpl)
  • Method 3: Download Annotation SOFT table manually

  1. Click the platform name GPL570 in Platforms (1) (Arrowed 1) in Figure 1 and scroll down to the bottom, as shown in Figure 2.

  2. Click Annotation SOFT table... (Arrowed in Figure 2) to start download.

  • The file is named GPL570.annot.gz end with a suffix of annot.gz.
  • Note to check the file size if download successfully.
  1. Import GPL570.annot.gz into R with getGEO{GEOquery}:
annot.gz <- getGEO(filename = "GPL570.annot.gz")
gpl <- annot.gz@dataTable@table
dim(gpl) # head(gpl), View(gpl), nrow(gpl)

Alternatively, fread{data.table} can be used as Step 3 in Method 2:

# install.packages("data.table")
library(data.table)
gpl <- fread("GPL570.annot.gz")
dim(gpl) # head(gpl), View(gpl), nrow(gpl)
Check the total number of rows of gpl
  • Method 4: Download SOFT formatted family file(s) manually
  1. Click SOFT formatted family file(s) in Download family (Arrowed 2) in Figure 1 and result looks alike in Figure 3.
  1. Click GSE50772_family.soft.gz (Arrowed in Figure 3) to download the file. It’s palced in the current working directory by default.
  2. Import GSE50772_family.soft.gz into R with getGEO{GEOquery}:
soft.gz <- getGEO(filename = "GSE50772_family.soft.gz")
gpl <- soft.gz@gpls[[platformID]]@dataTable@table
# gpl <- soft.gz@gpls$GPL570@dataTable@table
No matter which method you use, run code below to finish your work! 
save(gpl, file = paste0(gseID, ".GPL.Rdata"))

When conduct gene annotation later, we can load gpl by:

load(paste0(gseID, ".GPL.Rdata"))
