How to find and install R packages?

1. Find the right R packages we need

Every little makes a mickle. During daily work, we may come into useful R packages meeting our demand, as well as the newly developed ones. Usually, we can find most of basic and frquently used packages in the The Comprehensive R Archive Network (CRAN). As for people in the field of life science or fields-related to it, Bioconductor and Github may another two major repositories for R packages.

Apart from mentioned above, RDocumentation and Microsoft R Application Network (MRAN) may facilitate us to search for R packages. In MRAN, we can explore packages by Task Views.

2. Auto-install R packages of different sources

2.1 Packages from CRAN

# install.packages("packageNAME"), e.g., "ggplot2"
Install.packages("ggplot2")

2.2 Packages from Bioconductor

Packages hosted in Bioconductor are installed with BiocManager::install() other than install.packages(). So we need to install BiocManager first.

# install BiocManager first
# run only once for the first time
install.packages("BiocManager")

If packages from Bioconductor are needed, load BiocManager with library() and run codes below:

library(BiocManager)

# type the following in an R command window to install core packages, 
BiocManager::install()

# install specific packages, e.g., "GenomicFeatures" and "AnnotationDbi", with
# install one package
BiocManager::install("GenomicFeatures")

# use vector c() to install two or more packages
BiocManager::install(c("GenomicFeatures", "AnnotationDbi"))

2.3 Packages from GitHub

Installation guides are provied on Github by the author for packages hosted in Github.

For the installation of these packages, package devtools is requuired.

install.packages("devtools")

Demo with ClusterGVis hosted in Github.

library(devtools)

# note to update your ComplexHeatmap to the latest version!
devtools::install_github("junjunlab/ClusterGVis")

3 Install packages manually

3.1 Install package archive

If failed with methods above, try to download the package and install with RGui or RStudio.

  • For RGui
    • click Packages in navbar and select Install package(s) from local file.... Then browse for downloaded packages and install step-by-step.
  • For RStudio
    • Click Packages (Arrow 1 in Figure 1) in the section at the right-bottom followed by clicking install (Arrow 2 in Figure 1).
    • Install from option: change from Repository (CRAN) to Package Archive File (.zip; .tar.gz) (Arrow 1 in Figure 2).
    • Package archive option: click browse (Arrow 2 in Figure 2) to find the downloaded package and click Install finally.

3.2 Copy installed packages from others

At the very worst, you failed to install via the 4 ways just mentioned. What shall we do?

Don’t be disappointed 💔! To some extent, that is the easist case! 😄

Just copy the package installed from others who have installed successfully, and place it in the library folder under the installation directory of your R!
