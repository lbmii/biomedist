The R Series: R Markdown Cookbook

Yihui Xie, Christophe Dervieux, Emily Riederer
Tuesday, 20 October 2020
Chapman and Hall/CRC Press First Edition
R Markdown Cookbook: Cover

If you want to try the examples by yourself, the full source code of this book and examples are freely provided on GitHub. If you are reading the electronic version of this book, you may also just copy and paste the examples from the pages and run them in your favorite editor.

Despite the existence of the official documentation, R Markdown users often seek help on Stack Overflow, a popular Q&A forum. Authors have received comments from our readers and publisher that it would be beneficial to provide more practical and relatively short examples to show the interesting and useful usage of R Markdown, because it can be daunting to find out how to achieve a certain task from the aforementioned reference book (put another way, that book is too dry to read). As a result, R Markdown Cookbook was born. This book is designed to provide a range of examples on how to extend the functionality of your R Markdown documents. As a cookbook, this guide is recommended to new and intermediate R Markdown users who desire to enhance the efficiency of using R Markdown and also explore the power of R Markdown. The online version of R Markdown Cookbook is also free.
